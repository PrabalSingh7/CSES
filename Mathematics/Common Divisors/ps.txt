Common Divisors
TASK SUBMIT RESULTS STATISTICS HACKING
Time limit: 1.00 s Memory limit: 512 MB
You are given an array of n
 positive integers. Your task is to find two integers such that their greatest common divisor is as large as possible.

Input

The first input line has an integer n
: the size of the array.

The second line has n
 integers x1,x2,…,xn
: the contents of the array.

Output

Print the maximum greatest common divisor.

Constraints
2≤n≤2⋅105

1≤xi≤106

Example

Input:
5
3 14 15 7 9

Output:
7
